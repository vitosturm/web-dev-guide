Variables & Scope

Variables sind benannte Container für Werte. Scope bestimmt, wo im Code eine Variable sichtbar und zugänglich ist.

How it works
Animation Panel
📦
INFO 01

Deklaration mit let, const, var

Mit let und const deklarierst du Variablen mit Block-Scope. var ist veraltet und hat Function-Scope.

🔒
INFO 02

Block Scope

Jeder Block {} erzeugt einen neuen Scope. Variablen mit let/const sind außerhalb nicht sichtbar.

📈
INFO 03

Closure

Funktionen merken sich den Scope, in dem sie erstellt wurden — auch nach deren Ausführung.

🌍
INFO 04

Global Scope

Variablen außerhalb aller Funktionen und Blöcke sind global. Vermeide globale Variablen — sie können zu unerwarteten Konflikten führen.

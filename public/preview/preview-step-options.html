How It Works
🏷️

HTML is made of elements

Every visible thing on a webpage is an HTML element — a heading, a paragraph, a link, an image. Elements tell the browser what the content means, not just how it looks. The browser uses this meaning to render, style, and make the page accessible.

💡 Choose elements by meaning, not appearance — use CSS for how things look.
index.html
HTML 
<p>Hello, world!</p>
Result →Hello, world! (rendered as paragraph)
⚙️

Opening tag

The opening tag names the element. <p> means paragraph. <h1> means main heading. The name determines how the browser renders the content.

B — Notebook Persönlichster Charakter

Zweispaltiges Layout mit Emoji-Spalte links. Stärkstes visuelles Unterscheidungsmerkmal, warmes Lernheft-Gefühl.

C — Lesson Blocks Professionellste Optik

Top-Balken, Divider, "Example"-Label, Connector-Linien. React-Docs-Qualität — vertraut für Entwickler.

D — Tabbed Reveal Kompakteste Darstellung

Erklärung und Code in Tabs. Wenig Scrollbedarf — gut wenn Steps viel Inhalt haben.

E — Timeline Stärkstes Fortschrittsgefühl

Zeitlinie verbindet Steps. Bestes Gefühl für den eigenen Lernfortschritt — kein Opacity-Dimming.

Option C (Lesson Blocks) für professionelle Docs-Qualität mit dem klarsten Lernfluss.
Option A (Prose First) für sofortige Umsetzung ohne Layout-Umbau — nur das Dimmen entfernen.

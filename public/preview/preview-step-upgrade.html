04 — Vollständiges Upgrade (alle drei Bereiche kombiniert)
📦
INFO 01

var — der alte Weg

var ist function-scoped — nicht block-scoped. Eine Variable, die in einem if-Block deklariert wird, ist auch außerhalb sichtbar. Zusätzlich wird var an den Anfang seiner Funktion gehoisted.

Hoisting

JavaScript verschiebt var-Deklarationen an den Funktionsanfang — der Wert bleibt an Ort und Stelle. Das führt zu unerwartetem undefined bei frühem Zugriff.

Best Practice

Modernes JS: nutze let für veränderliche, const für unveränderliche Werte. var nur in Legacy-Projekten.

javascript 
var x = 1
if (true) {
  var x = 2   // gleiche Variable!
}
console.log(x)   // 2
